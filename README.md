
# Akıllı Müşteri Yönetim ve Analiz Sistemi

Bu proje, Python programlama dili kullanılarak Google Colab ortamında geliştirilen bir CRM (Müşteri İlişkileri Yönetimi) ve veri analizi sistemidir.

## Proje Amacı
Telekomünikasyon sektöründeki müşteri verilerini analiz ederek, müşteri segmentasyonu yapmak ve abonelikten ayrılma (churn) ihtimallerini tahmin etmek.

## Kullanılacak Teknolojiler
- **Dil:** Python
- **Ortam:** Google Colab
- **Kütüphaneler:** Pandas 
- **Veri Tabanı:** SQL (veya CSV tabanlı veri yönetimi)
